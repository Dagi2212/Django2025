# Week 3 – Python OOP Practice Tasks

This folder contains my solutions for **Week 3 – Python Object-Oriented Programming Practice Tasks**.  
Each problem is solved in its own Python file for clarity and proper organization.

---

## 📚 Overview

The tasks focus on the core concepts of OOP:

- Classes & Objects  
- Attributes (public & private)  
- Methods  
- Encapsulation  
- Inheritance  
- Method Overriding  

These exercises are foundational for the upcoming mini-projects.

---

## 📌 **List of Exercises**

### **1. Person Class**
- Create a `Person` class with `name` and `age`.
- Make an object and print its attributes.

### **2. Dog Class**
- Create a `Dog` class with a `bark()` method.
- Create an object and call the method.

### **3. Car Class**
- Class with attribute `make`.
- Add method `get_make()`.

### **4. Rectangle Class**
- Class with `width` and `height`.
- Print area directly using attributes.

### **5. Student Class (Private Attribute)**
- Private attribute `__grade`.
- Provide `set_grade()` and `get_grade()` methods.

### **6. Employee Class**
- Attributes: `name`, `salary`.
- Method: `annual_salary()`.

### **7. Library Class**
- Stores a list of books.
- Methods: `add_book()`, `show_books()`.

### **8. Animal & Cat (Inheritance)**
- Base class `Animal`, child class `Cat`.
- Both have a `make_sound()` method.

### **9. Vehicle → Car (Method Overriding)**
- Base class with `brand`, `year`.
- Child class adds `model`.
- Override `info()` method.

### **10. BankAccount Class**
- Private balance.
- Methods: `deposit()`, `withdraw()`, `get_balance()`.
- Prevents withdrawing more than the balance.
